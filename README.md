# FlutterBerita
Aplikasi Baca Berita

Data yang ditampilkan berasal dari newsapi.org.
Untuk mengakses dibutuhkan API Key yang di dapat setelah registrasi.
Url: https://newsapi.org/v2/top-headlines?country=id&apiKey=(Key)
